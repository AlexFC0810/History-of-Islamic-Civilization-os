# Visual Roadmap

## Purpose

Create the visual system for the Prophetic Age Seed OS.

The Golden Ages visuals showed the fruit. These visuals should show the seed.

## Style Direction

- Sacred but clear.
- Museum-quality, not social-media clutter.
- Warm ivory, deep navy, muted gold, soft earth tones.
- No depiction of the Prophet.
- Use abstract forms: light, calligraphy, map lines, architecture, dates, symbolic diagrams.
- Keep text editable in final Canva/Figma production.

## Flagship Visuals

### 1. Prophetic Age Clarity Map

**Core line:** Study the seed to understand the fruit.

**Sections:** soul, worship, character, family, market, knowledge, community, law, power, mercy.

### 2. Mecca to Medina Map

**Core line:** Mecca formed the inner world. Medina formed the public world.

**Structure:** Two-column formation map.

### 3. Seed to Fruit Map

**Core line:** The Golden Ages scaled what the Prophetic age seeded.

**Flow:** revelation -> knowledge -> character -> community -> law -> finance -> institutions -> Golden Ages.

### 4. Mercy as Architecture

**Core line:** Mercy was not only emotion. It became social architecture.

**Sections:** orphan, widow, poor, traveler, neighbor, servant, child, animal, enemy, ruler.

### 5. Moral Economy Map

**Core line:** Wealth was not rejected. It was placed under responsibility.

**Flow:** trade -> trust -> contracts -> giving -> inheritance -> public benefit -> endowments.

### 6. Knowledge Seed Map

**Core line:** The libraries began with knowledge becoming sacred.

**Flow:** revelation -> recitation -> memory -> teaching -> transmission -> books -> libraries.

### 7. Power Under Accountability

**Core line:** Power was not abolished. It was morally disciplined.

**Sections:** leadership, treaty, victory, restraint, judgment, responsibility.

### 8. Root to Fruit Masterboard

**Core line:** The Prophet's age is the root. The Golden Ages are the fruit.

**Structure:** root system below, civilizational tree above.
