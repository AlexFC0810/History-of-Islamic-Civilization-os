## Production Rule

Do not treat generated text as final. Rebuild every public visual with:

- editable text;
- source-backed claims;
- consistent typography;
- manually proofed spelling;
- citation notes in the design file;
- versioned exports.

## Suggested Final File Types

- `.fig` or Canva source link in a separate production tracker.
- `.png` export for social.
- `.pdf` export for archive.
- `.md` companion file with exact wording and sources.
