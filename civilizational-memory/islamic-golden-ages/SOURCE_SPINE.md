# Source Spine

## Purpose

This file defines the research-hardening workflow for the Islamic Golden Ages canon.

## Source Families

1. Academic encyclopedias and university references for translation, philosophy, medicine, mathematics, and institutional history.
2. Reference works for al-Khwarizmi, algebra, arithmetic, astronomy, and optics.
3. Medical history sources for bimaristans, hospitals, Ibn Sina, and al-Zahrawi.
4. Museum and art-history sources for geographic range, architecture, calligraphy, geometry, and material culture.
5. Economic-history sources for waqf, endowments, commerce, markets, public goods, and long-term funding.

## Claim Card Workflow

Every public-facing claim should include:

1. Claim
2. Source
3. Evidence grade
4. Caveat or counterclaim
5. Safe wording
6. Unsafe wording
7. Content angle
8. Visual angle

## Evidence Grades

- **A:** high confidence.
- **B:** strong but nuanced.
- **C:** useful hypothesis; needs more source work.
- **D:** do not publish yet.
